SQL SCHEMA DATABASE

UNTUK APA?

Digunakan saat ingin clone database ke project Supabase baru. Jalankan SQL ini terlebih dahulu di project baru, baru kemudian lakukan Restore dari file backup.

LANGKAH-LANGKAH
1

Buat project baru di supabase.com

2

Klik tombol di bawah → Copy SQL → Paste & Run di SQL Editor Supabase

3

Ganti URL & API Key di konfigurasi app ke project baru

4

Gunakan fitur Restore di bawah untuk isi data dari file backup

BACKUP & RESTORE DATA

TENTANG BACKUP

Backup mengekspor SEMUA data (master, transaksi, stok, pelunasan) ke file JSON. File ini dapat disimpan di HP dan digunakan untuk restore kapan saja.

DOWNLOAD BACKUP

Ekspor semua data ke file .json. Simpan di HP Anda sebagai cadangan.

RESTORE DARI BACKUP

Restore akan MENGHAPUS semua data saat ini dan menggantinya dengan data dari file backup. Proses ini tidak bisa dibatalkan.
